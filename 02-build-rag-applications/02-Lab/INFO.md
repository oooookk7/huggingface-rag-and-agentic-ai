### Getting Started

##### Run the Basic Gradio Demo

For simple calculations,

```
python3 gradio_demo.py
```

Or concatenating strings,

```
python3 gradio_demo2.py
```

Or the common input types overview,

```
python3 common_input_types.py
```

##### Run the LLM Chat Demo

```
python3 llm_chat.py
```

##### Run the LLM QA Demo

```
python3 qabot.py
```
